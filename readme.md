# Curso de Angular - FPF Tech

Instalação
instalar o nvm
instalar uma versão do node
node js
instalar o typescript
npm install -g typescript
instalar angular cli
npm install -g @angular/cli
verificar versao do angular
ng --version

usando typescript
compilar o arquivo
tsc nome_arquivo.ts
executar arquivo js
node nome_arquivo.js

usando angular
criar projeto
ng new nomeprojeto
